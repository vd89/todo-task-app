# todo-task-app
Todo task app with react


- steps to follow

  1. command to run create react app
    ```bash
      yarn create react-app . --template typescript
    ```
  2. Add the tailwind to the app
    ``` bash
      yarn add -D tailwind postcss autoprefixer
    ```
    ```bash
       npx tailwind init
    ```
    add
    ```css
    @tailwind base;
    @tailwind components;
    @tailwind utilities;
    ```
